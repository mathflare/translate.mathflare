# [Latin To Linear B Translator](https://translate.mathflare.xyz)

## Installation

1. Clone the repository
2. Install [Node.js](https://nodejs.org/en/)
3. Run `npm install`
4. Run `npm run dev`

## Technology Stack

- Nuxt.js 2.15
- Bootstrap.css 5.0
- Bootstrap.js 5.0

Deployed and protected by Cloudflare and Cloudflare Pages
